Mouse Event Properties

Related With Window & Mouse

  • clientX
    The clientX property returns the horizontal coordinate (according to the client area) of the mouse pointer when a mouse event was triggered.
    The client area is the current window.
  • clientY
    The clientY property returns the vetical coordinate (according to the client area) of the mouse pointer when a mouse event was triggered.
    The client area is the current window.



Related With Document & Mouse

  • pageX
    Returns the horizontal coordinate (according to the document)
    The document is the web page.
    Note: This property is non-standard, but works in most major browsers.
  • pageY
    Returns the vertical coordinate (according to the document)
    The document is the web page.
    Note: This property is non-standard, but works in most major browsers.



Related With Element & Mouse

  • offsetX
    Returns the horizontal coordinate relative to the position of the edge of the target element
  • offsetY
    Returns the vertical coordinate relative to the position of the edge of the target element



Related With Screen & Mouse

  • screenX
    Returns the horizontal coordinate (according to the users computer screen) of the mouse pointer when an event was triggered.
  • screenY
    Returns the vertical coordinate (according to the users computer screen) of the mouse pointer when an event was triggered.
