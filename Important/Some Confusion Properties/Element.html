Element Properties

Related With Height & Width

  • offsetHeight
    Returns the viewable height of an element, including padding, border and scrollbar not margin.
  • offsetWidth
    Returns the viewable width of an element, including padding, border and scrollbar not margin.


  • scrollHeight
    Returns the entire height of an element that is not viewable (because of overflow) in pixels, including padding , but not the border, scrollbar or margin.
  • scrollWidth
    Returns the entire width of an element that is not viewable (because of overflow) in pixels, including padding , but not the border, scrollbar or margin.


  • clientHeight
    The clientHeight property returns the viewable height of an element in pixels, including padding but not the border, scrollbar or margin.
  • clientWidth
    The clientWidth property returns the viewable width of an element in pixels, including padding but not the border, scrollbar or margin.




Related With Position

  • offsetParent :
    The offsetParent property returns the nearest ancestor that has a position other than static.
  • offsetleft :
    The offsetLeft property returns the left position (in pixels) relative to the left side the offsetParent element.
      The returned value includes :
    1. the left position, and margin of the element offsetParent element
    2. the left padding, scrollbar and border of the offsetParent element
  • offsetTop :
    The offsetTop property returns the top position (in pixels) relative to the top of the offsetParent element.
      The returned value includes :
    1. the top position, and margin of the element offsetParent element
    2. the top padding, scrollbar and border of the offsetParent element



Related with Scroll

  • scrollLeft
    The scrollLeft property sets or returns the number of pixels an element's content is scrolled horizontally.
  • scrollTop
    The scrollTop property sets or returns the number of pixels an element's content is scrolled vertically.
