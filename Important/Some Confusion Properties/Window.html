Window Properties

Related With Scroll

  • pageXOffset
    The pageXOffset property returns the pixels a document has scrolled from the upper left corner of the window.
    The pageXOffset property is equal to the scrollX property.
  • pageYOffset
    The pageXOffset property returns the pixels a document has scrolled from the upper left corner of the window.
    The pageXOffset property is equal to the scrollY property.
  • scrollX
    The scrollX property returns the pixels a document has scrolled from the upper left corner of the window.
    For cross-browser compatibility, use window.pageXOffset instead of window.scrollX.
  • scrollY
    The scrollY property returns the pixels a document has scrolled from the upper left corner of the window.
    For cross-browser compatibility, use window.pageYOffset instead of window.scrollY.




Related With Screen

  • screenLeft
    The screenLeft property returns the x (horizontal) position of a window, relative to the screen.
  • screenTop
    The screenTop property returns the y (vertical) position of the window relative to the screen.
  • screenX
    The screenLeft property returns the x (horizontal) coordinate of a window, relative to the screen.
  • screenY
    The screenTop property returns the y (vertical) coordinate of the window relative to the screen.

  • outerWidth

  • innerHeight
    The innerHeigth property returns the height of a window's content area.
  • innerWidth
