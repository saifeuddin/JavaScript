JavaScript Object Methods

Object is special data type where we can organize multiple data under a single variable. 

                

                    var obj = {
                        key : "value",
                        key : "value"
                    };
    # Object Methods
  1. Object.keys(object)

    Print object properties / keys inside an array

  2. Object.values(object)

    Print object properties values inside an array

  3. Object.entries(object)

    Returns a multidimensional / 2dimensional array for object

  4. Object.assign(object)

    Create a clone object
